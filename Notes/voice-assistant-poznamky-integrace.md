# Voice Assistant - Integrace poznámek

Testovací poznámka pro ověření funkčnosti hlasového zapisování.

## Plán implementace

- LLM Router rozpozná příkazy jako "Zapiš si, že..."
- NoteService uloží soubor do tohoto repozitáře
- Automatický git commit a push
- TTS potvrdí: "Poznámku jsem zapsal"

## Konvence pojmenování souborů

Soubory budou pojmenovány inteligentně podle obsahu:
- `nakupni-seznam.md` - pro nákupní seznamy
- `schuzka-s-klientem-2025-12-01.md` - pro poznámky ze schůzek
- `napad-nova-funkce-xyz.md` - pro nápady
